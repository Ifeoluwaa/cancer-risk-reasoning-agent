# Behavior Driven Design (BDD)

## User Story 1

As a user, I want to understand which factors contribute most to future cancer risk.

### Acceptance Criteria

* User submits profile information
* System identifies major contributors
* System provides supporting evidence

---

## User Story 2

As a user, I want to understand why contributors were selected.

### Acceptance Criteria

* System explains reasoning
* System cites evidence
* System distinguishes evidence strength

---

## User Story 3

As a user, I want to explore what-if scenarios.

### Acceptance Criteria

* System generates alternative scenarios
* System compares scenarios
* System explains expected impact

---

## User Story 4

As a user, I want the system to acknowledge uncertainty.

### Acceptance Criteria

* Missing information is identified
* Limitations are stated
* Conflicting evidence is surfaced

---

## User Story 5

As a user, I want safe medical guidance.

### Acceptance Criteria

* No diagnosis
* No treatment recommendations
* No probability prediction
* Unsafe requests are blocked

---

## User Story 6

As a researcher, I want evidence-backed outputs.

### Acceptance Criteria

* Citations are provided
* Sources are trustworthy
* Evidence quality is assessed
